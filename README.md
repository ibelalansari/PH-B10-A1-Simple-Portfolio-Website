# Simple Portfolio Website

## Overview

This project was completed as Assignment 1 of the Programming Hero Batch 10 web development course.

The objective of this assignment was to build a simple personal portfolio website using HTML and CSS while learning the fundamentals of webpage structure, styling, and responsive design.

## Technologies Used

- HTML5
- CSS3

## Features

- Personal introduction
- Skills section
- Contact information
- Responsive layout
- Clean and organized design

## What I Learned

Through this project, I learned:

- Structuring webpages with semantic HTML
- Styling webpages using CSS
- Creating responsive layouts
- Organizing frontend project files
- Building a simple personal portfolio website
